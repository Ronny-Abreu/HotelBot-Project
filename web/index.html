✦ Arquitectura

Así funciona
la magia

01

Azure Translator detecta

El mensaje del huésped llega y Azure Translator identifica el idioma en milisegundos y lo traduce al español para procesarlo.

02

GPT-4o procesa

Azure OpenAI con GPT-4o entiende la intención y genera una respuesta precisa usando el conocimiento del hotel.

03

Bot Service entrega

Azure Bot Service traduce la respuesta al idioma original y la entrega en menos de 2 segundos directamente al huésped.
